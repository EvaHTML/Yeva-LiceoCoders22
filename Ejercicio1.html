ETIQUETAS HTML

Cualquier cosa entre las etiquetas de apertura y cierre se interpretará como código HTML. Sin estas etiquetas, es posible que los navegadores interpreten incorrectamente su código HTML.

Algunas ejemplos de etiquetas :

  • br = ejecuta un salto de linea
  • ul = ejecuta una lista desordenada
  • li = se usa para describir una lista
  • h1 a h6 = proporciona titulos y subtitulos
  • pre = representa un texto preformateado
